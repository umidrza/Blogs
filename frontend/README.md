# Frontend

This is the frontend for the Blogs application, built with React, Vite, and Redux.

## Features

- User authentication
- Blog listing and creation
- User profiles
- Comments on blogs
- Responsive design with Bootstrap

## Technologies

- React 19
- Vite
- Redux Toolkit
- React Router
- Axios for API calls
- Vitest for testing
- ESLint for linting

## Installation

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

## Running the Application

- Development server:
  ```bash
  npm run dev
  ```

- Build for production:
  ```bash
  npm run build
  ```

- Preview production build:
  ```bash
  npm run preview
  ```

## Testing

Run tests:
```bash
npm test
```

## Linting

Check code style:
```bash
npm run lint
```

## Project Structure

- `src/components/` - React components
- `src/hooks/` - Custom hooks
- `src/reducers/` - Redux reducers
- `src/services/` - API service functions
- `src/store.js` - Redux store configuration
